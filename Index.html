How to Export CSV from MoTeC i2

This tool requires a MoTeC CSV export with distance data. Follow these steps in MoTeC i2 Pro:

Step 1 — Open your telemetry

  1. Open MoTeC i2 Pro and load your rFactor 2 telemetry file (.ld file from the rF2 log folder)
  2. In the Lap/Range selector (bottom of the screen), select the lap you want to export

Step 2 — Set up a Time/Distance graph

  1. Make sure you have a Time/Distance graph open with the channels you need. The default rF2 workspace usually includes all required channels
  2. The graph must be set to Distance mode (not Time) — click the T/D toggle in the graph toolbar if needed

Step 3 — Export as CSV

  1. Right-click on the Time/Distance graph
  2. Select Export Data...
  3. In the Export dialog:
    — Format: CSV
    — Data Extent: Selected Lap (or Zoomed Area if you want a specific section)
    — Sample Rate: 100 Hz recommended
    — Channels: All Channels (or Visible Channels if you have the right ones displayed)
    — Enable "Output Distance" — this checkbox adds the Lap Distance channel which is required for this tool
  4. Click Export and save the .csv file
Important: The CSV must contain the "Lap Distance" channel. If your export doesn't include it, add it to your graph before exporting. You can also include "GPS Latitude" and "GPS Longitude" for the track map visualization.

Required channels

The following channels are used (most are included by default): Ground Speed, Throttle Pos, Brake Pos, Gear, G Force Lat, Steering, GPS Latitude, GPS Longitude, Lap Number, Lap Distance, Session Elapsed Time, Path Lateral

Tips

  1. You can export two different laps from different sessions — each as its own CSV file — and compare them here
  2. You can also export multiple laps in one CSV file and select which laps to compare in the tool
  3. For rFactor 2, telemetry files are stored in: UserData\player\Log\
